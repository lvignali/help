You can require all users to fill a form before getting their tickets. This is specified in the Form tab of the Event details page.

For every field you can specify the name, the type and whether it is required or not. The list of types available are Checkbox, Date, E-mail, Number, Phone, Text (a single line of text), Textarea (a multi-line text field) and Select (a field where you can choose from a series of predefined values). Select fields can specify a list of values separated by commas.

When you are done, click Save and go get one ticket (View event → Get tickets). You will see that the purchasing process now includes an extra step:

Form report

To open the recorded data as a spreadsheet, go to your sold tickets list ( Event details → Select your performance → Sold tickets ) and click the Form Report button at the top. When prompted, give the file a .csv extension and save.

From your spreadsheet program (Excel, OpenOffice Calc) open the file and select the following:

  • Separator: commas
  • Character set: UTF-8

And click OK.
