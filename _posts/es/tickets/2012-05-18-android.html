Bienvenido a Koliseo Checkpoint Charlie

Koliseo Checkpoint Charlie es la aplicación Android gratuita para validar las entradas de tus eventos. Para usar esta aplicación debes tener una cuenta activa en Koliseo con eventos programados. Es importante que accedas a la aplicación con la misma cuenta que se utilizó para crear los eventos. Por el momento, la cuenta de Koliseo debe haberse abierto a través de una cuenta de Google.

Después de abrir sesión, la lista de eventos se sincronizará automáticamente con el servidor. Cuando este proceso haya finalizado podrás empezar a validar entradas. Recuerda realizar este paso al menos una vez antes de que se celebre el evento: Koliseo Checkpoint Charlie puede funcionar sin acceso a Internet, pero la lista de entradas debe haberse sincronizado con anterioridad.

Validación de entradas por localizador

Para validar mediante localizador, ve al listado de entradas y haz click en la barra de búsqueda. Al introducir el código el listado se actualizará automáticamente para reflejar tu entrada. Entonces, elige la entrada para mostrar sus detalles y hacer, si quieres, la validación.

Validación por código QR

También se pueden validar los códigos QR de las entradas mediante la cámara del dispositivo Android. Tan sólo enfoca el código QR con tu cámara.

Aunque se utilice Android para validar sigue siendo recomendable tener el listado de entradas vendidas en papel por si acaso.

¿Por qué el nombre?

Checkpoint Charlie era el punto de separación entre Berlin Oriental y Occidental durante la Guerra Fría. Era una fría noche de Marzo y estábamos buscando un nombre. Básicamente reprodujimos el complejo tren de pensamiento que ha hecho a otros llegar al número 42.
