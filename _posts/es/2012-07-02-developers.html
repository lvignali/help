Widgets

Puedes añadir la venta de entradas desde tus propias páginas con muy poco código. Estos widgets son de código libre y la documentación se encuentra disponible en GitHub.

Calendario

Un calendario que muestra las fechas de un evento.

Botón "Consigue tu entrada"

Una hoja de estilos CSS para el enlace para adquirir entradas desde tu página web.

REST API

El API de Koliseo está disponible para uso comercial o no comercial por terceras partes en cualquier tipo de integración. La documentación del API es un proceso en marcha en la actualidad, y únicamente las funcionalidades que se consideran estables son publicadas. Todo lo que no se mencione en la documentación no debe ser considerado final.

La documentación del API está disponible en Mashape.
